## O que é esta integração

Importação CSV é a forma mais comum de trazer base histórica de planilhas, outro CRM ou listas de marketing para o Mercato — desde que você trate o arquivo como **contrato de dados**, não como “atalho mágico”.

[INSERIR IMAGEM: tela de importação CSV com preview]

## Quando usar

- Migração inicial de contatos
- Campanhas com listas segmentadas (com governança)
- Atualização em lote quando não há integração API disponível

## Benefícios

- Velocidade para enriquecer o funil com histórico
- Controle sobre o que entra (por colunas mapeadas)
- Possibilidade de importação incremental por lotes

  <Step title="Modelagem: defina campos mínimos">

  </Step>
  <Step title="Higiene: padronize e deduplique">

## Observações

<Tip>
  Se você não consegue explicar a origem de uma coluna, ela provavelmente não deveria entrar no primeiro import.
</Tip>
